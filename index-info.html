🎯 Smart Resource Booking System

A modern, fully-functional web application for booking resources with role-based access and admin priority system

🚀 Quick Start (3 Steps)

  1. Click "Start Application" above to open the login page
  2. Enter any email and copy the generated OTP shown in the alert
  3. Verify OTP and select your role (User or Admin) to access the dashboard

👥 User Roles & Permissions

👤 User Role
  • View dashboard & statistics
  • Book rooms (Meeting Rooms, etc.)
  • View personal bookings
  • Cancel own bookings
  • See availability calendar
  • Receive booking confirmation
⚙️ Admin Role
  • All User features PLUS:
  • View all system bookings
  • Override user bookings
  • Delete any booking
  • Manage resources
  • View booking analytics
  • Handle double bookings

🎯 How It Works

Booking Process

Step Action Result
1 Select room, date, time Form validation
2 Check for double bookings Prevent duplicates or show override option
3 Create booking record Save to localStorage
4 Update availability Show slot as booked
5 Show success message Toast notification

Admin Override Process

Step Action Result
1 Admin tries to book occupied slot Override confirmation modal shown
2 Admin confirms override Previous booking marked "Overridden"
3 New admin booking created Slot now assigned to admin
4 Update all views User and admin see changes
5 Show priority message Both users notified via toast
